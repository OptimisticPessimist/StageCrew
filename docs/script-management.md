# 脚本管理機能 要件まとめ

pscweb3 にあった脚本管理機能のうち、StageCrew に取り込むべき機能を整理したもの。

---

## 1. 脚本データモデル

### Script（脚本）
| フィールド | 説明 |
|-----------|------|
| id | 一意識別子 |
| production_id | 所属公演 |
| title | 脚本タイトル |
| content | 脚本本文 |
| uploaded_by | アップロードしたユーザー |
| uploaded_at | アップロード日時 |
| author | 脚本家名 |
| draft_date | 脱稿日 |
| revision | 改訂番号 |
| revision_text | 改訂メモ |
| copyright | 著作権表示 |
| contact | 連絡先 |
| notes | 備考 |

### PDF出力設定
| フィールド | 説明 |
|-----------|------|
| pdf_orientation | 用紙方向（横/縦） |
| pdf_writing_direction | 文字方向（縦書き/横書き） |

### 公開設定
| フィールド | 説明 |
|-----------|------|
| is_public | 公開/非公開 |
| public_terms | 公開利用条件 |
| public_contact | 公開用連絡先 |

---

## 2. シーン・セリフ・登場人物

### Scene（シーン）
- act_number（幕番号）、scene_number（場番号）による階層構造
- heading（見出し）、description（説明）

### Character（登場人物）
- name（名前）、description（説明）、order（表示順）

### Line（セリフ）
- content（内容）、order（表示順）
- character_id（発言者）、scene_id（所属シーン）

---

## 3. 脚本アップロード・パース

### 対応フォーマット
- **Fountain形式**: メタデータ自動抽出、登場人物・シーン見出しの自動解析
- **その他（プレーンテキスト / PDF / Word 等）**: ファイルアップロード＋手動でシーン・登場人物を登録

### Fountain形式の自動解析
- 登場人物セクション（`# 登場人物`）の解析
- シーン見出し検出（INT./EXT. 形式やカスタム形式）
- あらすじ（Scene #0）の処理
- マルチエンコーディング対応（UTF-8, Shift_JIS等）

---

## 4. キャスティング（配役）

- 登場人物へのキャスト割当
- ダブルキャスト対応（1役に複数キャスト）
- 表示名・キャストメモの管理
- 脚本更新時にキャスティング情報を保持

---

## 5. 香盤表（Scene Chart）

- シーンごとの登場人物出番マッピング
- どの役者がどのシーンに出るか一覧表示
- 稽古・本番のスケジュール管理に活用
- **Fountain形式の場合**: パース結果から自動生成
- **その他フォーマットの場合**: 手動でシーン・登場人物を登録し、手動編集

---

## 6. PDF出力

- 縦書き/横書き対応
- 日本語フォント対応
- メタデータ付き（著者、日付、改訂番号、連絡先）
- 脚本フォーマットに沿ったレイアウト

---

## 7. 公開・共有機能

- 脚本の公開/非公開切り替え
- 公開脚本の一覧表示（ページネーション対応）
- 公開脚本のインポート（他公演へ取り込み）
- 公開利用条件・連絡先の設定

---

## 8. Discord通知連携

- 脚本アップロード/更新時にDiscord Webhook通知
- 改訂番号・公演名・脚本タイトルを含む
- 生成PDFの添付

---

## 9. バージョン管理

- 改訂番号（revision）による版管理
- 改訂テキスト（revision_text）による改訂メモ
- 更新時に関連データ（シーン、セリフ、マッピング）を再構築
- キャスティング・稽古データは更新時も保持
